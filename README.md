1.Main libraries

```
torch==1.1.0
cuda==10.0
libtiff==0.4.2
```

2.Training

generate label json file
```
python create_data_lists.py
```

then

```
python train.py
```
Note that due to samll scale of training dataset, the mAP may have relatively large jitters (about 2 mAP) with different random seeds.

3.Eval

```
python eval.py
```
